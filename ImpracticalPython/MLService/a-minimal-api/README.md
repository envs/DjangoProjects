# A Minimal API
This API will return a "Hello World" JSON object


## Usage
1. Run the API
```python
python api.py
```

2. Open a new terminal window and run the following:
```bash
curl http://127.0.0.1:5000/
```

3. This will output the following JSON object:
```
{"hello": "world"}
```
